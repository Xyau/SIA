Pasos para correr el script bash:
	-Generar el archivo de configuración .m ("están como ejemplo config.m config2.m config3.m)
	- Correr el comando "source script.sh x" siendo x el nombre del archivo .m
	- Al finalizar el script quedará en la terminal de octave, teniendo como opción usar la red neuronal manualmente a través de "p=ans", donde p es la instancia de la clase Perceptrón.
